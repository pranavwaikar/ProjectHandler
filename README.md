# Project handler
A project management tool for teams. 
This is a MERN stack application 

# Pre-requistes
- Node - 14.18.0

# How to Run ? 

1. Install dependencies: 
````
npm install
cd client
npm install
cd ..
````

2. Run both frontend and backend servcies - 
````
npm run dev
````
